# Narrative Flow (Numeric 1-10)

## What to evaluate

Does the analysis tell a coherent story? Are sections logically ordered (context -> method -> findings -> implications)? Are transitions smooth? Does each section build on the previous one?

## Scoring Anchors

- **9-10:** Reads like a compelling story; each section flows naturally to the next
- **7-8:** Good structure; minor transition gaps
- **5-6:** Sections feel somewhat disconnected; reader has to work to follow
- **3-4:** No clear narrative arc; sections appear in arbitrary order
- **1-2:** Stream-of-consciousness; no discernible structure

## What moves the score up

- Clear through-line from question to answer
- Transition sentences linking each section to the next
- Evidence presented before conclusions (show the work, then state the claim)
- Each section builds on information established earlier

## What moves the score down

- Conclusions stated before supporting evidence
- Abrupt section transitions with no linking sentence
- Repeating information across sections
- Topics introduced without connection to the central question
